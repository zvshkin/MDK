ВАРИАНТ № А15/Б12

1. Даны ненулевые числа x, y. Проверить истинность высказывания: «Точка с координатами (x, y) лежит в первой координатной четверти».

2. Дано расстояние L в сантиметрах. Найти количество полных метров в нем (1 метр = 100 см).

3. Дан целочисленный массив из N элементов. Найти количество элементов после последнего максимального элемента.

4. Вводится строка слов, разделенных подчеркиваниями. Определить количество слов, начинающихся и заканчивающихся на одну букву.

5. Вводится строка слов, разделенных подчеркиваниями. Определить количество слов, у которых второй и последний символ одинаковые.
