ВАРИАНТ № А15/Б12

1. Проверить истинность высказывания: "Квадратное уравнение A·x² + B·x + C = 0 с данными коэффициентами A (A не равно 0), B, C имеет ровно один вещественный корень".

2. Дан целочисленный массив, состоящий из N элементов (N > 0). После каждого отрицательного элемента массива вставить элемент с нулевым значением.

3. Вводится строка, состоящая из слов, разделенных подчеркиваниями. Определить количество слов, которые начинаются и заканчиваются одной и той же буквой.

4. Вводится строка вида «цифра_цифра_цифра_цифра», где «_» - это «+» или «-». Вывести значение выражения.

5. Вводится строка с буквами и цифрами. Вывести сумму и произведение цифр строки.
